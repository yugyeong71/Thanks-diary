[[emailVerify]]
== 3. 이메일 인증 코드 검증
*이메일 인증 코드 검증 API*

: 인증 코드가 만료되기 전에 잘못된 인증 코드를 요청하면, `인증 코드 불일치` 오류가 발생합니다.

인증 코드가 만료된 후에는 인증 코드의 일치 여부와 관계없이, `인증 코드 만료` 오류가 발생합니다.

*[ 이메일 인증 코드 검증 실패 ]*::

- 이메일 미입력 : `"이메일은 필수입니다."` (code : 417)
- 올바르지 않은 이메일 형식 : `"올바르지 않은 이메일입니다."` (code : 417)
- 인증 코드 미입력 : `"인증 코드는 필수입니다."` (code : 417)
- 인증 코드 만료 : `"인증 코드가 만료되었습니다."` (code : 401)
- 인증 코드 불일치 : `"인증 코드가 일치하지 않습니다."` (code : 401)
